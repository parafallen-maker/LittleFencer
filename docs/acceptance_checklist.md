# 🤺 LittleFencer P2.1 实地测试验收清单

> **目标**: 验证全佩剑动作检测算法在真实场景下的准确性与灵敏度。
> **环境**: 明亮室内，手机横屏放置于地面或三脚架 (约0.5米高)，距离 2.5-3米。

---

## 1. 基础姿态 (En Garde)

| 测试动作 | 预期反馈 | 判定标准 |
|:---|:---|:---|
| **标准站姿** | ✅ 只有绿色骨架，无语音 | 前膝~110°，后腿微屈，重心平稳 |
| **测试错误：蹲得太低** | 🔊 "Too low!" | 前膝角度 < 90° |
| **测试错误：站得太直** | 🔊 "Bend more!" | 前膝角度 > 135° |
| **测试错误：后腿完全弯曲** | 🔊 "Push back leg!" | 后腿角度太小 (像坐着一样) |
| **测试：头部低垂** | 🔊 "Head up!" | 鼻子位置显著低于肩部中线 |

## 2. 基础步法 (Footwork)

| 测试动作 | 预期反馈 | 判定标准 |
|:---|:---|:---|
| **标准前进步 (Advance)** | 🔊 "Good advance!" | 保持姿态，平稳向前移动约 30-50cm |
| **标准后退步 (Retreat)** | 🔊 "Good retreat!" | 保持姿态，爆发式向后移动 |
| **测试：快速连续前进** | 连续识别或合并 | 至少识别出一次完整的 Advance |
| **错误测试：带跳跃的前进** | ❌ 无反馈 或 误判 | 身体起伏过大应不被识别为标准 Advance |

## 3. 进攻动作 (Attacks) - *核心测试*

| 测试动作 | 预期反馈 | 判定标准 |
|:---|:---|:---|
| **标准弓步 (Lunge)** | 🔊 "Nice lunge!" | 手臂先伸 + 后腿蹬直 + 前脚落地 |
| **测试错误：先动脚后伸手** | 🔊 "Arm first!" | 这是新手最常见的错误，必须敏锐捕捉 |
| **测试错误：膝盖超过脚尖** | 🔊 "Knee forward!" | 保护膝盖的关键检测 |
| **前进弓步 (Advance-Lunge)** | 🔊 "Nice attack!" | 前进步接弓步，中间无明显停顿 |
| **错误测试：剑尖下垂** | 🔊 "Blade level!" | 攻击时手腕过低 |

## 4. 高级动作 (Advanced)

| 测试动作 | 预期反馈 | 判定标准 |
|:---|:---|:---|
| **两步起跳弓步 (Balestra)** | 🔊 "Nice Balestra!" | 明显的双脚腾空跳跃，落地后立即弓步 |
| **飞弓步 (Flunge)** | 🔊 "Flunge!" | 极高速度前冲，身体前倾，**后脚未过前脚** |
| **错误测试：冲刺越线** | 🔊 "Back foot passed!" | 在做 Flunge 时，故意让后脚超过前脚 |

## 5. 防守反击 (Defense)

| 测试动作 | 预期反馈 | 判定标准 |
|:---|:---|:---|
| **格挡 (Parry) - 外侧** | 🔊 "Good parry!" | 手腕向外侧快速移动 (Tierce) |
| **格挡 (Parry) - 头顶** | 🔊 "Good parry!" | 手腕快速上抬 (Quinte) |
| **格挡反攻 (Parry-Riposte)** | 🔊 "Nice riposte!" | 格挡后 0.5秒内快速刺出 |
| **测试：乱挥剑** | ❌ 无反馈 | 无规律的晃动不应触发 Parry |

---

## 📝 记录注意事项

- **光线**: 确保没有强背光，身体轮廓清晰。
- **干扰**: 画面中最好只有一名击剑者。
- **确认音**: 每次有效动作后应有 "Ding" 声或语音。
- **误报率**: 记录在静止或做其他动作时是否误报了 Lunge/Parry。
